# 🏁 DSA Race Leaderboard

A lightweight leaderboard that gamifies **Data Structures & Algorithms** practice by ranking participants based on their progress. Built to encourage consistency, accountability, and healthy competition among friends preparing for coding interviews.

### 🌐 Live Demo

https://udhay090.github.io/DSA-Race-Leaderboard-/

## ✨ Features

* Dynamic leaderboard ranking
* Progress tracking
* Responsive UI
* Fast, lightweight, and fully client-side
* Built with HTML, CSS, and JavaScript
* Hosted on GitHub Pages

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript
* GitHub Pages

## 🚀 Future Improvements

* Automatic LeetCode profile sync
* Daily streak tracking
* Difficulty-wise statistics
* Contest performance
* Achievement badges
* Dark mode
